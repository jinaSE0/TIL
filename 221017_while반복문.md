<h2>while 반복문</h2>

while(조건){

}

```
1. 불 표현식을 확인한다
2. true이면 본문을 실행하고 처음으로 돌아간다
3. false이면 종료한다
```
for와 while 반복문을 고르는 법
```
while반복문은 조건이 중요할때 
예를들면
- 결과가 나올 때까지
- 특정 시간이 될 때까지
- 파일을 읽으며 특정 단어를 찾을 때까지
for반복문은 특정 횟수만큼 반복을 돌거나 배열을 기반으로 반복을 돌때 사용함
```
반복문에 break나 continue를 활용할 수 있는데
continue를 활용하면 아래의 코드로 내려가지않고 계속 반복문으로 돌아가게됨
