Application
---
```
Command Palette (Mac : ⇧⌘P, F1 / Windows : Ctrl + Shift + P, F1)
Quick Open (Mac : ⌘P / Windows : Ctrl + P)
User Setting (Mac : ⌘ / Windows : Ctrl+)
Toggle Sidebar (Mac : ⌘B / Windows : Ctrl+B)
Toggle Terminal (Mac : ⌃` / Windows : Ctrl+`)
Keyboard Shortcuts (Mac : ⌘+K+S / Windows : Ctrl+K+S)
```
 
Editor
---
```
파일의 맨 위/ 아래 이동 (Mac : ⌘↑/⌘↓ / Windows : Ctrl+Home/Ctrl+End)
문장의 처음/끝 이동 (Mac : ⌘←/⌘→ / Windows : Home/End))
단어별 이동 (Mac : ⌥←/→ / Windows : Ctrl+←/→)
한 단어 선택 (Mac : ⌥⇧←/→ / Windows : Ctrl+Shift+←/→)
라인 복사 (Mac : ⌘C / Windows : Ctrl+C)
라인 잘라내기 (Mac : ⌘X / Windows : Ctrl+X)
라인(블럭) 위 아래 이동 (Mac : ⌥↑/⌥↓ / Windows : Alt+↑/↓)
다음 줄 코드 작성 (Mac : ⌘Enter / Windows : Ctrl+Enter)
반복되는 같은 단어 선택해서 한번에 수정하기 (Mac : ⌘D / Windows : Ctrl+D)
이전 커서의 위치로 이동하기 (Mac : ⌘U / Windows : Ctrl+U)
다중선택 커서넣기 (Mac : ⌥+Click / Windows : Alt+Click)
같은 줄 선택(커서넣기) (Mac : ⇧⌥+drag mouse / Windows : Shift+Alt+drag mouse)
선택 블럭 끝에 커서 넣기 (Mac : ⇧⌥i / Windows : Shift+Alt+i)
```
