<h2>다양한 메서드</h2>

- 배열에서 최댓값찾을때 조심하자
```
Math.max(51,40,30)//51
Math.max([51,40,30])//배열을 이렇게 넣으면 안됨
Math.max(...[51,40,30])//이렇게 넣어야 올바르게 출력됨
```
- 0과 1사이의 난수를 만들어줌
```
Math.random()
```

- 0과 50사이의 랜덤한 정수는 어떻게 만들까
```
Math.floor(Math.random()*50)
```
